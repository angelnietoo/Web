Ejemplo Básico: Contador de Palabras

Un Map es ideal para contar la cantidad de veces que aparece un elemento en una colección.

const frase = 'hola mundo js js';
const palabras = frase.split(' ');
const contador = new Map();
palabras.forEach(p => contador.set(p, (contador.get(p) || 0) + 1));
console.log(contador);

// clone corto y convertir
